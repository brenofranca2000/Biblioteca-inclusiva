

Breno França Magrani# Biblioteca Inclusiva API

API REST desenvolvida com Spring Boot para gerenciamento de usuários, categorias, livros e empréstimos de uma biblioteca inclusiva.

---

##  Tecnologias Utilizadas

- Java 17
- Spring Boot
- PostgreSQL
- Maven
- Swagger/OpenAPI
- JPA/Hibernate
- Git e GitHub

---

## Estrutura do Projeto

O projeto foi organizado utilizando:

- Controller
- Service
- Repository
- DTO
- Entity

---

## Funcionalidades

### Usuários
- Cadastrar usuário
- Listar usuários
- Buscar usuário por ID

### Categorias
- Cadastrar categoria
- Listar categorias
- Buscar categoria por ID

### Livros
- Cadastrar livro
- Listar livros
- Buscar livro por ID

### Empréstimos
- Realizar empréstimo
- Listar empréstimos
- Buscar empréstimo por ID
- Deletar empréstimo

---

## Banco de Dados

O projeto utiliza PostgreSQL com integração via Spring Data JPA.

---

## Documentação Swagger

Após iniciar o projeto, acesse:

http://localhost:8080/swagger-ui/index.html

---

## Como Executar

1. Clonar o repositório
2. Configurar o PostgreSQL
3. Executar o projeto no IntelliJ
4. Acessar o Swagger
